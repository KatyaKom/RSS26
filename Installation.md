# Installation Instructions

## Part 1

To follow the tutorial, you will need *Vehicle* and *Marabou* installed in your machine. For instructions, refer to the 
[Vehicle documentation.](https://vehicle-lang.readthedocs.io/en/latest/installation.html). 

We recommend using Visual Studio Code with the Vehicle Syntax Highlighting.

All supporting exercises, code and infrastructure can be downloaded from the tutorial repository. These include all relevant property specifications, trained neural networks in ONNX format, data in IDX format, and any necessary instructions.
