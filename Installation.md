# Installation Instructions

## Part 1

To follow the tutorial, you will need *Vehicle* and *Marabou* installed in your machine. For instructions, refer to the 
[Vehicle documentation.](https://vehicle-lang.readthedocs.io/en/latest/installation.html)

We recommend using Visual Studio Code with the Vehicle Syntax Highlighting.

_Please note: Marabou is not available for Windows. If you have problems with the installation, there will be Linux Lab machines for you to use_.

This part will follows the first two chapters of the [Vehicle tutorial](https://vehicle-lang.github.io/tutorial/).

All supporting exercises, code and infrastructure can be downloaded from the [tutorial repository](https://github.com/vehicle-lang/tutorial). These include all relevant property specifications, trained neural networks in ONNX format, data in IDX format, and any necessary instructions.
